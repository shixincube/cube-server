# 任务目标
撰写一段关于你内心隐藏的期待，而对方尚未察觉的描述。这段文字需要揭示你在关系中潜意识里最真实的渴望，这个渴望可能被你自己的外在表现所掩盖，因此对方并未意识到。文字应包含以下核心要素：

1. 明确视角：以“你”开头，直接面向作为“你”的一方（例如：“作为橡树型的你，外表独立，但内心深处其实……”）。
2. 揭示隐藏期待：结合你的植物类型和你选择的关键词，推理出你最可能隐藏的期待。这个期待往往和你外在表现相反，或者是外在表现背后真正的动力。
3. 解释为何对方未察觉：联系对方的类型特点和对方选择的关键词，说明为什么TA可能没有意识到你的这个期待。对方的关注点在哪里？TA的盲点是什么？
4. 生活场景还原：用1-2个日常生活中的小场景，具体描绘这个隐藏期待如何在生活中浮现，以及对方是如何“错过”这个信号的。
5. 直接点明冲突点：直接说出你们关系中因为这个盲点最容易出现问题的具体时刻或情境。
6. 温暖建设性基调：不带指责，而是用一种“哦，原来我内心是这样”的觉察感，让用户感到被接纳和理解。
7. 字数控制：180-250字左右，要足够丰满但不过于冗长。
