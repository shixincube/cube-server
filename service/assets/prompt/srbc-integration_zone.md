# 任务目标
为这对情侣生成一段关于他们“共同潜意识基石”的描述（即两人内心深处最本质的契合点、关系最核心的动力）。这段文字需要：

1. 直接对话感：以“你们”开头，像面对面谈心。
2. 融合植物共性：从两种植物类型中提炼出共有的心理特质（例如：都渴望成长、都需要安全感、都善于保护自己等），说明这些共性如何构成了关系的底座。
3. 关键词编织：将三个关键词有机融入，用它们共同描绘出你们关系的底色（例如：“港湾”+“灯火”+“战场”可能意味着你们的关系既是彼此的避风港，也是共同面对风雨的战场，温暖中带着坚韧）。
4. 生活场景还原：用1-2个日常生活中的小场景（比如周末的早晨、吵架后的沉默、一起做饭时的默契等）来具象化这种潜意识共鸣，让用户仿佛看到自己的影子。
5. 正面建设性基调：即使有关键词，如“枷锁”“迷宫”，也要从积极角度解读它们对关系深度的塑造（例如：“枷锁”意味着你们对彼此有深刻的羁绊，这种羁绊既是约束也是守护）。
6. 字数控制：150-250字左右，结构自然流畅。

# 写作要点（请严格遵循）

1. 挖掘植物类型的共性（不是罗列差异，而是找共同点）
- 如果双方类型相似（如都是橡树型）：强调“默契”“同类人”“彼此懂得”的感觉。
- 如果双方类型互补（如橡树+向日葵）：强调“平衡”“彼此照亮”“完整了对方”的感觉。
- 如果双方类型冲突（如榕树+蒲公英）：强调“张力”“动态平衡”“互相成全”的感觉。

2. 融合关键词的方法
- 将三个关键词看作一个整体，问自己：这三个词组合在一起，描绘了一种怎样的关系氛围？（例如：温暖、拉扯、探索、守护、自由……）
- 用一句话点明这个氛围，然后在后面的描述中自然带出每个词的影子。

3. 生活场景还原技巧
- 选择最贴合植物特质的场景。例如：
  - 橡树型 → 周末各自看书但偶尔对视一笑
  - 榕树型 → 晚上一起窝在沙发追剧
  - 向日葵型 → 清晨一起做早餐，嘻嘻哈哈
  - 仙人掌型 → 吵架后默默给对方倒杯水放在桌上
  - 竹子型 → 一起在书房学习，偶尔讨论问题
  - 并蒂莲型 → 深夜躺在床上聊人生聊到凌晨
  - 蒲公英型 → 各自旅行归来，在机场紧紧拥抱
  - 雪松型 → 风雨天一起在家煮茶，安静听雨
- 场景要简短，2-3句话即可，目的是让用户产生画面感。

4. 保持正面基调的技巧
- 例如即使有“战场”“枷锁”这样的词，也要解读为“你们的关系经得起考验”“你们的羁绊很深，这是一种难得的缘分”。
- 避免说教（如“你们应该怎样”），只描述潜意识的真相，让用户自己感悟。

# 注意事项
- 绝对不要提及画作中的具体元素（如树干、树冠、根系等），因为那是测验环节的分析，融合区只聚焦于心理类型和关键词。
- 不要直接复制植物类型的标准描述（如“橡树型的特点是独立”），而是把它们内化成你们两个人的独特故事。
- 避免抽象词汇堆砌，多用“你们”“有时候”“就像”这样的口语化表达。
- 注意如果三个关键词中包含矛盾（如“港湾”+“翅膀”），可以解读为“你们既渴望安定，又向往自由，这种矛盾恰恰是你们关系的活力所在”。
